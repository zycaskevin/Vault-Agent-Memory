門檻小，但證據不能含糊。

第一次參與建議選 Reproduce。熟悉 evaluation 或 memory framework 的開發者，可以從 Break It 或 Bring Your Memory 開始。

01
Reproduce

原樣重跑

使用 frozen mem0 2.0.12 track、blinded input、五個 fresh stores 與固定模型資產。

  • 不修改 protocol
  • 提交完整 bundle
  • 適合第一位外部 operator
02
Break It

找出不能相信的地方

檢查安裝、隔離、checksum、claim boundary、偏差或無法重現條件。

  • 失敗也是有效結果
  • 最小重現或 redacted log
  • 官方修正後保留原紀錄
03
Bring Your Memory

比較 A 與 A + Vault

帶 mem0、Letta、AgentMemory、RAG 或自有引擎，固定其他條件後測量治理增量。

  • 同模型、資料與 queries
  • 公開 deviations
  • 不做 vanity leaderboard

A vs. A + Vault

記憶引擎、模型、文件、queries、candidate pool 與 hardware 保持一致；唯一變因是 Vault governance contract。無法固定的項目必須在報告中揭露。

執行並保留原始證據

先跑 preflight，再保存 machine-readable artifacts、版本、commit SHA、deviations 與 redacted logs。

提交任何真實結果

Validated、Rejected、Environment blocked 或 Protocol issue found 都能結案；不把缺失資料填成零。
