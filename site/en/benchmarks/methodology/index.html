Reproducible methodology

Fix the question first. Then publish the answer and its limits.

Every headline is generated from a versioned, checksummed publication bundle. The website formats evidence; it does not recalculate or hand-enter results.

Protocol

Scope

Six-case synthetic controlled retrieval-only evaluation; top-1 with candidate pool 4.

Repeats

Five sequential clean-state process repeats for each track.

Pairing

Compare the same engine as A and A + Vault; report latency only as a paired delta.

Publication gates

If checksums, manifests, repeat counts, or publishable gates fail, no headline is shown.

Reproduce it

python scripts/build_benchmark_site_data.py
python -m pytest tests/test_benchmark_site.py

See the complete fixture and provider contract in the external memory benchmark documentation.

Claim boundary

This evidence measures a retrieval governance contract. It is not end-to-end answer quality, an official LoCoMo / LongMemEval leaderboard, or a zero score for frameworks that have not been measured.
