為什麼是現在

Agent 增長得比記憶信任基礎設施更快。

真正的平台機會不是再做一個封閉記憶庫，而是建立能橫跨 Agent stack 的治理合約。

每個 Agent 都變成寫入者

沒有 candidate-first 控制，自主寫入會讓共享記憶變成未經審查的真實來源。

每個框架都形成孤島

團隊需要跨 Codex、Claude、OpenClaw、n8n、Coze 的連續性，而不是另一座 proprietary island。

信任成為基礎設施

權限、來源、過期、回滾和審計是水平需求，這正是 Vault 要擁有的平台層。

護城河

能穿越 backend 變動的治理合約。

模型、向量資料庫與記憶引擎會不斷更換；可信記憶的治理不變量不會消失。

Provider-neutral

同一治理合約跨本機、自架、雲端與未來 managed backend。

Evidence-native

Bounded reads、引用、審計與 digest-bound publication artifacts。

Lifecycle-aware

推廣、過期、取代、封存、刪除、還原與回滾。

Agent-native

MCP、Gateway、CLI、OpenAPI 與受治理的跨主機同步。
