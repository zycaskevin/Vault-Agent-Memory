Reproducible methodology

先固定問題，再公開答案與限制。

所有 headline 都由版本化、checksummed publication bundle 產生。網站只格式化，不重新計算或手填結果。

Protocol

範圍

6-case synthetic controlled retrieval-only；top-1，candidate pool 4。

重複

每個 track 五次 sequential clean-state process repeats。

配對

比較同一引擎 A 與 A + Vault；latency 只報 paired delta。

發布閘門

checksums、manifest、重複數與 publishable gate 任一失敗，就不顯示 headline。

如何重現

python scripts/build_benchmark_site_data.py
python -m pytest tests/test_benchmark_site.py

完整 fixture 與 provider contract：external memory benchmark 文件。

Claim boundary

這項證據測量 retrieval governance contract，不代表 end-to-end answer quality，不是官方 LoCoMo / LongMemEval 排行，也不將尚未測量的框架記為零分。
