Preflight

先用機器可讀檢查，避免跑到一半才失敗。

python scripts/run_external_reproduction.py \
  --output-dir /tmp/vault-external-reproduction \
  --model-cache-dir ../vault-repro-model-cache \
  --preflight-only --json

一次檢查 clean source、外部 venv/output、固定套件、磁碟、記憶體、模型 cache 與 Hugging Face 連線。exit 2 會列出所有 blocked checks；pass 只表示環境就緒。

主張邊界

Preflight pass

只代表環境符合執行條件，不是 benchmark 或第三方重跑結果。

Contract validated

才代表固定 synthetic retrieval-governance contract 在揭露環境中成功重現；仍不是普遍優越性、production scale 或端到端 QA。

證據階梯

Validation 與 endorsement 明確分開。
