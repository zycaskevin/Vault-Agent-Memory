# Vault-for-LLM v0.6.117

This release continues the module-size cleanup by moving automation feedback-learning helpers into `vault.automation_learning`.

The daily automation loop keeps the same behavior: learning still only adjusts review priority and health reporting. It does not authorize auto-promotion, deletion, archive, cold-store, or active-memory writes.

This makes the automation core easier to audit while preserving the agent-friendly automation cycle, inbox, brief, handoff, learning-health, and fleet-health workflows.
