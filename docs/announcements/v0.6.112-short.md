# v0.6.112

This release continues the module-size cleanup from v0.6.110 and v0.6.111.

## Changed

- Moved automation report, handoff, and Markdown artifact helpers into
  `vault.automation_reports`.
- Reduced `vault/automation.py` from 4747 lines to 3946 lines.
- Added `vault/automation_reports.py` at 834 lines, below the default
  new-module module-size gate.
- Lowered the `vault/automation.py` module-size baseline so it cannot grow back
  silently.

## Compatibility

Public automation behavior is unchanged. Existing CLI commands and MCP tools
continue to use the same names and payloads.
