# Vault Agent Memory v0.10.2

v0.10.2 adds a safe way to check whether Vault has a newer release.

## What changed

- `vault upgrade --check` compares the installed version with PyPI.
- `vault upgrade` currently performs the same check-only flow.
- Vault detects common pip, pipx, uv tool, and editable source installs and
  prints the appropriate upgrade command.
- `--latest-version` supports offline comparisons.
- `--json` and `--pretty` expose the result to Agents and automation.

## Safety boundary

This release does not auto-upgrade Vault. The command never invokes a package
manager, changes the Python environment, or migrates `vault.db`. Review and run
the recommended command yourself, then verify with:

```bash
vault --version
vault doctor
```

## Install

```bash
pip install --upgrade "vault-for-llm[mcp,supabase]==0.10.2"
```

After installation:

```bash
vault --version
vault upgrade --check
```

Expected version: `vault-for-llm 0.10.2`.
